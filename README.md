##InvoiceFlow: Autonomous International Invoicing

Architected a custom AI-driven solution to automate the extraction and validation of global financial documents.

Stack: Python, Django/Flask, LLM Integration (RAG-based parsing), and Vector Embeddings.

Innovation: Moving beyond simple OCR by implementing NLP-driven "Rule Maps" that allow the system to adapt to different international languages and legal requirements without manual reconfiguration.

Focus: High-output, low-latency data pipelines that replace manual labor with proprietary code.
